GitHub Actions Setup

Add this to your .github/workflows/*.yml file

- name: Notify Telegram
  if: always()
  run: |
    curl -X POST https://hooks.zapier.com/hooks/catch/25470556/uzws3gf/ \
    -H "Content-Type: application/json" \
    -d '{"status": "${{ job.status }}"}'

Add if: always() to ensure notification runs even if previous steps fail

GitLab CI Setup

Add this to your .gitlab-ci.yml file

after_script:
  - |
    curl -X POST https://hooks.zapier.com/hooks/catch/25470556/uzws3gf/ \
    -H "Content-Type: application/json" \
    -d "{\"status\": \"$CI_JOB_STATUS\"}"

Use after_script to ensure notification runs regardless of job status
